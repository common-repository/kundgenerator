The plugin is aimed at those who use Kundgenerator+ (http://www.kundgenerator.se) and wish to add logging to their Wordpress site.

== Description ==

Integrating Kundgenerator+'s web code on your website. You will need your login information for Kundgenerator+. If you do not have administrative permissions in Kundgenerator+, contact us and we will help you.

The web code will be located in the footer.php but will be invisible for visitors.

== Installation ==

1. Upload `kundgenerator.php` to the `/wp-content/plugins/` directory, with kundgenerator.js too.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You will be sent to the Kundgenerator+'s website where you need to login and choose settings. 
4. Save the settings.

== Frequently asked questions ==

= Will the plugin change the look of my website? =

No, the plugin includes only web code on your site.
